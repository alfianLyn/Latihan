Cara membuat sebuah artikel sederhana

1. Buat sebuah struktur html standar.

Bila anda menggunakan laptop buatlah sebuah struktur html pada notepad, anda bisa mengetikkan struktur dasar html seperti yang ada di gambar bawah. Bila anda menggunakan SoloLearn anda bisa memilih menu "{ }" setelah itu pilih "+" dan pilih Web.

2. Membuat sebuah judul.

Gunakan tag atau tanda <h1> untuk membuat judul dan <h3> untuk membuat sub judul letakkan pada tag <body>.
Silahkan lihat contoh.

3. Membuat sebuah paragraf atau deskripsi.

Untuk membuat paragraf atau deskripsi gunakan tag atau tanda <p> sama layaknya membuat judul letakkan tag tersebut dalam tag body, bila ingin membuat baris baru pada sebuah paragraf gunakan tag <br>
Silahkan lihat contoh.

3. Membuat tulisan pada paragraf menjadi tebal, miring atau garis bawah.

Seperti pada microsoft word pada HTML juga dapat membuat tulisan menjadi tebal, miring atau memberi garis bawah pada tulisan caranya adalah

  • Tag <b> : untuk membuat tulisan menjadi tebal atau bold
  • Tag <i> : untuk membuat tulisan menjadi miring atau italic
  • Tag <u> : untuk memberi garis bawah pada tulisan atau underline
tag tersebut nantinya akan disisipkan dikata atau kalimat yang ingin kita rubah.
Silahkan lihat contoh.

4. Membuat list.

Untuk membuat list pada sebuah paragraf dapat menggunakan tag <ul> (membuat list dalam bentuk bullet) atau <ol> (membuat list dalam bentuk angka) dan diikuti tag <li>
Silahkan lihat contoh.

Dari langkah diatas akan diperoleh hasil seperti ini.
